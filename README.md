# The Syntax Surgeon 🩺💻

## AI-Powered Code Fixer & Conversational Assistant

The Syntax Surgeon is an intelligent AI chatbot that can detect, analyze, and fix Python code errors while also providing conversational assistance. It uses the IBM Granite 3.3 2B language model to generate responses and help developers debug their code.

---

## 🚀 Features

* AI Chatbot for programming questions
* Automatic Python code detection
* Syntax error detection using Python AST
* Automatic code formatting and fixing
* Conversation history tracking
* Download chat history as TXT or PDF
* Gradio-based interactive web interface

---

## 🛠 Technologies Used

* Python
* IBM Granite 3.3 2B Instruct Model
* Hugging Face Transformers
* PyTorch
* Gradio
* Black Formatter
* autopep8
* ReportLab
